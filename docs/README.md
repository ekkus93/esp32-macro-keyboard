# Project Documentation

This directory contains the design, architecture, specifications, implementation plans, and reference material for the ESP32 Macro Keyboard project.

## Planned contents

- `mockups/` — web application screen mockups and a screen index
- architecture and subsystem design documents
- macro-set, procedure, and macro data schemas
- HTTP API specification
- firmware and web application implementation specifications
- testing, security, recovery, and release documentation

The documentation should remain synchronized with the implemented firmware and web application behavior.
